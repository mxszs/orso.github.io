JavaScript异步编程

May 24, 2018

js是一门单线程语言，异步编程对js语言来说就显得尤为重要

现在实现异步编程的方式基本上有四种

  • 回调函数
  • Promise函数
  • Generator函数
  • async/await
回调函数
Promise函数
Generator函数
async/await

异步函数存在以下四种使用形式：

  1. 函数声明： async function foo() {}
  2. 函数表达式： const foo = async function() {}
  3. 对象的方式： let obj = { async foo() {} }
  4. 箭头函数： const foo = async () => {}
